## Getting started

Scaffold a new project using [degit](https://github.com/Rich-Harris/degit):

```sh
npx degit ibm/carbon-components-svelte#webpack svelte-app
cd svelte-app
yarn install
```

## Available scripts

### `yarn dev`

Starts the app in development mode.

### `yarn build`

Builds the app for production.
